<html><head></head><body><p>Every week I come across some great articles. Here are some of the posts I feel you will enjoy reading as much as I did.</p>
<p>Wondering if <a href="rent-buy-house">you should buy or rent</a>? Luckily there is a tool that can help answer such questions for you. <a href="https://www.causal.app/">Casual</a> by <strong>Taimur Abdaal</strong> and <strong>Lukas Koebis</strong> is a tool that allows accounting for uncertainty when building models and forecasts.</p>
<p>You have probably heard that SQL Injection is a common attack vector for infiltrating a web application. <strong>Omer Hamerman</strong> shows <a href="https://omerxx.com/sql-injection-intro/">how a hacker can use this vulnerability to attack your application</a>.</p>
<p><strong>Aaron Xie</strong> shows how you would approach <a href="https://www.educative.io/blog/system-design-tinyurl-instagram">designing services like TinyURL and Instagram</a> from a system perspective.</p>
<p><strong>David Perell</strong> tells you <a href="https://threadreaderapp.com/thread/1192474076360654850.html">what lessons you can take from different fields</a>.</p>
<p>Serilog is an excellent logging library for .Net. <strong>Ben Foster</strong> presents some <a href="https://ben.onfabrik.com/blog/serilog-logging-best-practices">tips and best practices using Serilog</a> in your web applications.a</p>
<p><strong>Steve Smith</strong> shares the <a href="https://ardalis.com/configuring-a-local-test-email-server">different options to run a local test email server</a> so that you can validate that your code runs without actually sending emails.</p>
<p>Hope you enjoy reading these articles as much as I did.</p>
