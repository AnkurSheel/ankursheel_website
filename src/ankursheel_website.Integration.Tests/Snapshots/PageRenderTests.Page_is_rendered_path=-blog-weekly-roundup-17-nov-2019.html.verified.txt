<html><head></head><body><p>Every week I come across some great articles. Here are some of the posts I feel you will enjoy reading as much as I did.</p>
<p><strong>Martin Gasston</strong> has some great advice on <a href="https://8thlight.com/blog/martin-gaston/2019/10/22/tips-for-technical-writing.html">writing technical articles</a> at the 8th Light blog. He also has some creative writing exercises to help you along. If you are looking for a template, he also has that covered.</p>
<p><a href="https://bennettfeely.com/clippy/"><strong>Clippy – CSS clip-path Maker</strong></a> is a handy tool (made by <strong>Bennet Feely</strong>) to create shapes using the <code>clip-path</code> CSS property. It also comes with some predefined shapes.</p>
<p>How do the most popular testing tools for React compare to each other? <strong>Sunil Sandhu</strong> tests the same app using Jest, Enzyme, Testing Library, and Cypress.Check it out <a href="https://medium.com/javascript-in-plain-english/i-tested-a-react-app-with-jest-testing-library-and-cypress-here-are-the-differences-3192eae03850">here</a></p>
<p>Ever wondered what type of developer you are? <strong>Neil Green</strong> has a <a href="https://www.12types.dev/quiz">handy quiz</a>. My top trait was <em>The Assassin</em>: capable of quickly and quietly finishing their work before anyone even knows they have started. I think this will also be a suitable tool to ask your coworkers to fill for you.</p>
<p><strong>Neal Agarwal</strong> has a fantastic tour of scale within the known universe at <a href="https://neal.fun/size-of-space/">The Size of Space</a>.</p>
<p>Hope you enjoy reading these articles as much as I did.</p>
