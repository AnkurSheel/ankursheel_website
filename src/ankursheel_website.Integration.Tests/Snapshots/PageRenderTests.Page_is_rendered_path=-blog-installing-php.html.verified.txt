<html><head></head><body><h2 id="installing-php-on-windows">Installing PHP on Windows</h2>
<ul>
<li>Go to the PHP website and download the zip file.</li>
<li>Make sure the zip file contains the dll for the correct version of the Apache.</li>
<li>Extract the zip file to a location that does not contain spaces.</li>
</ul>
<h2 id="configuring-apache-for-php">Configuring APACHE for PHP</h2>
<p>Open httpd.conf for editing. You can access this file through the menu at Start -&gt; Programs -&gt; Apache HTTPD Server -&gt; Configure Apache Server -&gt; Edit the Apache httpd.conf File</p>
<ul>
<li><p>Configure Apache to load the PHP module.</p>
<ul>
<li>Find the <em><strong>LoadModule</strong></em> section.</li>
<li>Add the following line to <em>LoadModule</em>_ php5<em>module</em> “[path to php5apache2_4.dl in the PHP folder]”. This would be something Like <em>"D:/Softwares/php/php5apache2</em>_<em>4.dll"</em>. Make sure that you use forward slashes.</li>
</ul>
</li>
<li><p>Configure Apache to recognize PHP extensions.</p>
<ul>
<li>Find the <em><strong>AddType</strong></em> section.</li>
<li>Add the following line <em>AddType application/x-httpd-php .php</em></li>
</ul>
</li>
<li><p>Restart the APACHE server.</p>
</li>
</ul>
<p>If you see errors, check the Apache error.log</p>
<h2 id="configuring-php">Configuring PHP</h2>
<ul>
<li>Copy <em>php.ini</em> to the Windows directory.</li>
<li>Open <em>php.ini</em> for editing.</li>
<li>Set your local timezone.
<ul>
<li>Find this line; date.timezone = and remove the semicolon in front of it.</li>
<li>Find the timezone code from Php timezones and add the code for the timezone after the = sign.</li>
</ul>
</li>
<li>Restart the APACHE server.</li>
</ul>
