<html><head></head><body><h2 id="problem">Problem</h2>
<p>How do you upload a PDF file generated by a 3rd party API to an AWS S3 bucket and protect it using server-side encryption?</p>
<h2 id="solution">Solution</h2>
<p>Let's assume that we have got the pdf as a stream from calling our API endpoint.</p>
<pre><code class="language-csharp hljs">Stream pdfStream = api.GetPdfAsStream();
</code></pre>
<h3 id="snippet">Snippet</h3>
<p>Let's see the code before talking about what is happening.</p>
<pre><code class="language-csharp hljs"><span class="hljs-keyword">var</span> s3Client = <span class="hljs-keyword">new</span> AmazonS3Client(RegionEndpoint.USEast1);
<span class="hljs-keyword">var</span> s3Request = <span class="hljs-keyword">new</span> PutObjectRequest
{
    BucketName = <span class="hljs-string">"Bucket Name"</span>,
    Key = Guid.NewGuid().ToString(),
    InputStream = pdfStream,
    ServerSideEncryptionMethod = ServerSideEncryptionMethod.AWSKMS,
    ContentType = <span class="hljs-string">"application/pdf"</span>,
    Headers = { ContentDisposition = <span class="hljs-string">$"attachment; \"filename=\"File Name\""</span> },
};

<span class="hljs-keyword">var</span> s3Response = <span class="hljs-keyword">await</span> s3Client.PutObjectAsync(s3Request);
</code></pre>
<ul>
<li><strong><em>Line 1</em></strong>: Setup the S3 client with the region we want to connect to</li>
<li><strong><em>Line 2</em></strong>: Create a new <code>PutObjectRequest</code> because we want to upload a file to S3.</li>
<li><strong><em>Line 4</em></strong>: <strong><em>BucketName</em></strong>: Set the bucket name to which the PDF needs to be uploaded.</li>
<li><strong><em>Line 5</em></strong>: <strong><em>Key</em></strong>: Create a unique key to identify the object in S3.</li>
<li><strong><em>Line 6</em></strong>: <strong><em>InputStream</em></strong>: Set the inputStream to the PDF stream we received from the API.</li>
<li><strong><em>Line 7</em></strong>: <strong><em>ServerSideEncryptionMethod</em></strong>: Use the AWS Key Management Service to encrypt the PDF.</li>
<li><strong><em>Line 8</em></strong>: <strong><em>ContentType</em></strong>: Set the content type to <em>application/pdf</em> so the browser can show the file as a PDF.</li>
<li><strong><em>Line 9</em></strong>: <strong><em>Headers</em></strong>: Add some common headers
<ul>
<li><em>ContentDisposition</em> to <em>attachment</em> to indicate that the file should be downloaded.</li>
<li><em>filename</em> to the name of the file.</li>
</ul>
</li>
<li><strong><em>Line 12</em></strong>: Start the asynchronous execution of the <code>PutObject</code> operation.</li>
</ul>
<h2 id="conclusion">Conclusion</h2>
<p>We can upload a PDF to an S3 bucket using this snippet and encrypt it using server-side encryption.</p>
