<html><head></head><body><h2 id="installing-phpmyadmin-on-windows">Installing phpMyAdmin on Windows</h2>
<ul>
<li>Go to the phpMyAdmin Website and download the zip file</li>
<li>Unpack the downloaded file to the web server Document root.</li>
<li><em>Optional:</em> Rename the directory.</li>
<li>Create a copy of the file <em>config.sample.inc.php</em> and rename it to <em>config.inc.php</em></li>
<li>Open config.inc.php for editing and add/update the following
<ul>
<li><em>$cfg['Server'][$i]['user'] = 'root';</em></li>
<li><em>$cfg['Server'][$i]['password'] = 'root-password'</em></li>
</ul>
</li>
</ul>
<h2 id="configuring-the-web-server-for-phpmyadmin">Configuring the web server for phpMyAdmin</h2>
<ul>
<li>Open the <em>httpd.conf</em> file(located in the conf folder of the webserver) for editing</li>
<li>Find <em>DirectoryIndex</em> and add <em>index.php</em>. This will look something like this
<ul>
<li><em>DirectoryIndex index.html index.php</em></li>
</ul>
</li>
</ul>
<h2 id="configuring-php-for-phpmyadmin">Configuring php for phpMyAdmin</h2>
<ul>
<li>Open <em>php.ini</em> (in the System Directory) for editing</li>
<li>Remove the semicolon from the line <em>;extension=php_mbstring.dll</em>
<ul>
<li>If the line does not exist, add the line.</li>
</ul>
</li>
</ul>
<h2 id="testing-phpmyadmin">Testing phpMyAdmin</h2>
<ul>
<li>Restart the server</li>
<li>Navigate to <em>http://localhost:8080/phpMyAdmin/</em> on the browser.</li>
<li>You should see the phpMyAdmin login window</li>
<li>Enter the username and password in the login window.</li>
<li>You should now see the phpMyAdmin Control panel</li>
</ul>
