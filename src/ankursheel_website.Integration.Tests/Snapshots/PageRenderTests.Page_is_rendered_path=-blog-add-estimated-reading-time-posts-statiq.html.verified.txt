<p><em>You can install <a href="https://www.nuget.org/packages/Codinators.StatiqHelpers">StatiqHelpers as a Nuget Package</a> if you just want a module to add to your website.</em></p>
<p>In this tutorial, I will show how to use the reading time calculated in <a href="calculate-reading-time">How to calculate reading time</a> to use in a Statiq module to add reading time for our posts.</p>
<p>To add a module to Statiq, we can inherit from one of the provided base module classes. In this case, we can inherit from the <code>ParallelModule</code> class as the engine can process the documents in any order. We also need to overload the <code>ExecuteInputAsync()</code> method to execute this module for each document in the pipeline.</p>
<pre><code class="language-csharp hljs"><span class="hljs-keyword">public</span> <span class="hljs-keyword">class</span> <span class="hljs-title">GenerateReadingTime</span> : <span class="hljs-title">ParallelModule</span>
{
    <span class="hljs-keyword">protected</span> <span class="hljs-keyword">override</span> <span class="hljs-keyword">async</span> Task&lt;IEnumerable&lt;IDocument&gt;&gt; ExecuteInputAsync(IDocument input, IExecutionContext context)
    {
        <span class="hljs-keyword">using</span> <span class="hljs-keyword">var</span> textReader = input.GetContentTextReader();
        <span class="hljs-keyword">var</span> content = <span class="hljs-keyword">await</span> textReader.ReadToEndAsync();

        <span class="hljs-keyword">return</span> input.Clone(
                <span class="hljs-keyword">new</span> MetadataItems
                {
                    { MetaDataKeys.ReadingTime, _readingTimeService.GetReadingTime(content, context.GetInt(&quot;ReadingTimeWordsPerMinute&quot;, <span class="hljs-number">200</span>)) }
                })
            .Yield();
    }
}
</code></pre>
<ul>
<li><strong><em>Line 5-6</em></strong>: Get the content associated with the document as a string.</li>
<li><strong><em>Line 8</em></strong>: Clone the document by adding new metadata items.</li>
<li><strong><em>Line 10-11</em></strong>: Add a new metadata item with key <em>Reading_time_and a value obtained from a call to <code>GetReadingTime</code>. We use a _ReadingTimeWordsPerMinute</em> setting to override the default value of 200 words per minute.</li>
<li><strong><em>Line 11</em></strong>: Convert the document into an enumerable.</li>
</ul>
