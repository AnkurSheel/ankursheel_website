<html><head></head><body><p>I created <a href="https://www.nuget.org/packages/Codinators.XunitHelpers/">Codinators.XunitHelpers</a>(a NuGet package) to add some useful unit testing functionality that is missing in xUnit.</p>
<p>This release adds support for reporting multiple assertion failures in one method in xUnit tests similar to how <a href="https://github.com/nunit/docs/wiki/Multiple-Asserts">Nunit does it</a>.</p>
<h2 id="installation">Installation</h2>
<p>Install it by using the following command</p>
<pre><code class="language-bash hljs">dotnet add package Codinators.XunitHelpers
</code></pre>
<h2 id="usage">Usage</h2>
<pre><code class="language-csharp hljs">[Fact]
<span class="hljs-function"><span class="hljs-keyword">public</span> <span class="hljs-keyword">void</span> <span class="hljs-title">MultiAssertTest</span>(<span class="hljs-params"></span>)
</span>{
    <span class="hljs-keyword">var</span> result = <span class="hljs-number">2</span> + <span class="hljs-number">2</span>;
    AssertHelper.AssertMultiple(() =&gt; Assert.IsType&lt;<span class="hljs-keyword">double</span>&gt;(result),
                                () =&gt; Assert.Equal(<span class="hljs-number">5</span>, result),
                                () =&gt; Assert.Equal(<span class="hljs-number">4</span>, result));
}
</code></pre>
