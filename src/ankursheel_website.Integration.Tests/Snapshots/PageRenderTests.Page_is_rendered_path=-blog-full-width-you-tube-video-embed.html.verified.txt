<html><head></head><body><p>I struggled for a long time to get my YouTube videos full-width with a 16:9 aspect ratio. Then I found a neat CSS trick to make iframes responsive.</p>
<p>But before we look at the code, let's look at the before and after screenshots.</p>
<p><img src="/assets/images/posts/full-width-you-tube-video-embed/before.jpg" alt="Before"></p>
<p><img src="/assets/images/posts/full-width-you-tube-video-embed/after.jpg" alt="After"></p>
<h2 id="steps">Steps</h2>
<p>Let's use CSS to display an iframe with 100% width and 16:9 aspect ratio.</p>
<p>When embedding a video from YouTube, we get a snippet of code similar to</p>
<pre><code class="language-html hljs"><span class="hljs-tag">&lt;<span class="hljs-name">iframe</span>
    <span class="hljs-attr">width</span>=<span class="hljs-string">"560"</span>
    <span class="hljs-attr">height</span>=<span class="hljs-string">"315"</span>
    <span class="hljs-attr">src</span>=<span class="hljs-string">"https://www.youtube.com/embed/RcnksOUugcA"</span>
    <span class="hljs-attr">frameborder</span>=<span class="hljs-string">"0"</span>
    <span class="hljs-attr">allow</span>=<span class="hljs-string">"accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"</span>
    <span class="hljs-attr">allowfullscreen</span>
&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-name">iframe</span>&gt;</span>
</code></pre>
<h3 id="step-1">Step 1</h3>
<p>Remove all unnecessary properties, including <em><strong>width</strong></em> and <em><strong>height</strong></em>. Our snippet now becomes</p>
<pre><code class="language-html hljs"><span class="hljs-tag">&lt;<span class="hljs-name">iframe</span> <span class="hljs-attr">src</span>=<span class="hljs-string">"https://www.youtube.com/embed/RcnksOUugcA"</span> <span class="hljs-attr">allowfullscreen</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-name">iframe</span>&gt;</span>
</code></pre>
<h3 id="step-2">Step 2</h3>
<ol>
<li>Add a container with a CSS class around the iframe.</li>
<li>Add a CSS class to the iframe.</li>
</ol>
<pre><code class="language-html hljs"><span class="hljs-tag">&lt;<span class="hljs-name">div</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"video-container"</span>&gt;</span>
    <span class="hljs-tag">&lt;<span class="hljs-name">iframe</span> <span class="hljs-attr">class</span>=<span class="hljs-string">"video"</span> <span class="hljs-attr">src</span>=<span class="hljs-string">"https://www.youtube.com/embed/RcnksOUugcA"</span> <span class="hljs-attr">allowfullscreen</span>&gt;</span><span class="hljs-tag">&lt;/<span class="hljs-name">iframe</span>&gt;</span>
<span class="hljs-tag">&lt;/<span class="hljs-name">div</span>&gt;</span>
</code></pre>
<h3 id="step-3">Step 3</h3>
<p>Add CSS to the container.</p>
<pre><code class="language-CSS hljs"><span class="hljs-selector-class">.video-container</span> {
  <span class="hljs-attribute">position</span>: relative;
  <span class="hljs-attribute">width</span>: <span class="hljs-number">100%</span>;
  <span class="hljs-attribute">padding-bottom</span>: <span class="hljs-number">56.25%</span>;
}
</code></pre>
<p><strong>What are we doing here?</strong></p>
<ul>
<li><strong><em>position: relative</em></strong>: Lets us use absolute positioning for the iframe.</li>
<li><strong><em>width: 100%</em></strong>: Make the width 100% of its parent container.</li>
<li><strong><em>padding-bottom: 56.25%</em></strong>: The 16:9 aspect ratio corresponds to a height that is 56.25% of the width.</li>
</ul>
<p><strong><em>Note</em></strong>: To find the container aspect ratio, use this formula: <em>height ÷ width = aspect ratio</em>.</p>
<h3 id="step-4">Step 4</h3>
<p>Add CSS to the iframe.</p>
<pre><code class="language-CSS hljs"><span class="hljs-selector-class">.video</span> {
  <span class="hljs-attribute">position</span>: absolute;
  <span class="hljs-attribute">top</span>: <span class="hljs-number">0</span>;
  <span class="hljs-attribute">left</span>: <span class="hljs-number">0</span>;
  <span class="hljs-attribute">width</span>: <span class="hljs-number">100%</span>;
  <span class="hljs-attribute">height</span>: <span class="hljs-number">100%</span>;
  <span class="hljs-attribute">border</span>: <span class="hljs-number">0</span>;
}
</code></pre>
<p><strong>What are we doing here?</strong></p>
<ul>
<li><strong><em>position: absolute</em></strong>: Free the video from the height boundary of its parent and allow it to be positioned over the padding area.</li>
<li><strong><em>top: 0</em></strong>: Position the video at the top of its parent container.</li>
<li><strong><em>left: 0</em></strong>: Position the video at the left of its parent container.</li>
<li><strong><em>width: 100%</em></strong>: Stretch the video to fill the width of its parent container.</li>
<li><strong><em>height: 100%</em></strong>: Stretch the video to fill the height of its parent container.</li>
<li><strong><em>border: 0</em></strong>: Remove the border.</li>
</ul>
