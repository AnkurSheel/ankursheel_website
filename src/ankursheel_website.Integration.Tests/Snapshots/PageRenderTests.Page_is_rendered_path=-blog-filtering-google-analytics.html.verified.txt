    <h1 class="text-center">How to filter yourself from showing up in Google Analytics</h1>

<html><head></head><body><p>Google Analytics is a powerful and useful tool for anybody with any website who wants to track their traffic.</p>
<p>I will attempt to show exactly how to add analytics and filter yourself from the data if you have a self-hosted blog using WordPress.</p>
<h2 id="adding-analytics">Adding Analytics</h2>
<ol>
<li><p>Sign in to your Analytics account, and click Admin in the top menu bar.</p>
</li>
<li><p>Select the <em>account</em> and <em>property</em> in the respective columns.</p>
</li>
<li><p>Add tracking to your website.</p>
</li>
<li><p>Create a new profile on Google Analytics (if you have not already created one).</p>
</li>
<li><p>Add the tracking code snippet to your website.</p>
</li>
<li><p>Click Tracking Info under the Property column.</p>
</li>
<li><p>Copy the <strong>tracking code</strong> snippet.</p>
</li>
<li><p>Now, you have 3 options:</p>
<ol>
<li><strong>Not Recommended:</strong> Modify header.php and paste the tracking code (from step 2.2) before the <code>&lt;/head&gt;</code> tag. This method is <strong>NOT</strong> recommended because if the theme is updated, you will lose your changes.</li>
<li>Use the <strong>Insert Headers and Footers plugin</strong> and add it under the <strong>Scripts in header</strong> tab.</li>
<li>Modify functions.php in your <strong>child theme</strong> and paste the following code.</li>
</ol>
<pre><code class="language-php hljs"><span class="hljs-function"><span class="hljs-keyword">function</span> <span class="hljs-title">sp_google_analytics_tracking_code</span><span class="hljs-params">()</span> </span>{
    ?&gt;
    &lt;script type=&quot;text/javascript&quot;&gt;
        <span class="hljs-keyword">var</span> _gaq = _gaq || [];
        _gaq.push([<span class="hljs-string">'_setAccount'</span>, <span class="hljs-string">'UA-45876690-1'</span>]);
        _gaq.push([<span class="hljs-string">'_trackPageview'</span>]);
            (<span class="hljs-function"><span class="hljs-keyword">function</span><span class="hljs-params">()</span> </span>{
        <span class="hljs-keyword">var</span> ga = document.createElement(<span class="hljs-string">'script'</span>); ga.type = <span class="hljs-string">'text/javascript'</span>; ga.async = <span class="hljs-keyword">true</span>;
        ga.src = (<span class="hljs-string">'https:'</span> == document.location.protocol ? <span class="hljs-string">'https://ssl'</span> : <span class="hljs-string">'http://www'</span>) + <span class="hljs-string">'.google-analytics.com/ga.js'</span>;
        <span class="hljs-keyword">var</span> s = document.getElementsByTagName(<span class="hljs-string">'script'</span>)[<span class="hljs-number">0</span>]; s.parentNode.insertBefore(ga, s);
        })();
    &lt;/script&gt;
    &lt;?php
}

<span class="hljs-comment">// include GA tracking code before the closing head tag</span>
add_action(<span class="hljs-string">'wp_head'</span>, <span class="hljs-string">'sp_google_analytics_tracking_code'</span>);
</code></pre>
</li>
</ol>
<h2 id="creating-the-filter-in-google-analytics">Creating the Filter in Google Analytics</h2>
<ol>
<li>On the Google Analytics admin page, click on <strong>Filter</strong> under the View option. Here is where you create your exclusion filter.</li>
<li>Click on <strong>&quot;New Filter&quot;</strong>.</li>
<li>Give it a meaningful name.</li>
<li>In the <strong>Filter Type</strong>, select <strong><em>Custom Filter</em></strong> and leave it on the default <em>Exclude</em>.</li>
<li>In the <strong>Filter Field</strong>, select <em>User Defined</em>.</li>
<li>In the <strong>Filter Pattern</strong>, add no_report.</li>
<li>Save the filter.</li>
</ol>
<h2 id="get-your-google-analytics-account-id">Get your Google Analytics Account ID</h2>
<ol>
<li>Click <em>Property Settings</em> under the property column.</li>
<li>Copy the code under <strong>Tracking ID</strong>. This will look something like <em>UA-XXXXXXXX-X.</em>.</li>
</ol>
<h2 id="create-an-html-page-to-create-a-cookie-excluder">Create an Html page to create a cookie excluder</h2>
<p>Create a new Html file and paste the following code.</p>
<pre><code class="language-html hljs">&lt;meta http-equiv=&quot;”Content-Type”&quot; content=&quot;”text/html;&quot; charset=&quot;iso-8859-1?&quot; /&gt;google analytics cookie excluder
&lt;!-- prevents indexing of this page by search engines --&gt;&lt;meta name=&quot;ROBOTS&quot; content=&quot;NOINDEX,NOFOLLOW&quot; /&gt;&lt;script
    type=&quot;text/javascript&quot;
&gt;
    // &lt;![CDATA[
    var _gaq = _gaq || [];
    _gaq.push(['_setAccount', '&lt;em&gt;&lt;strong&gt;UA-XXXXXXXX-X&lt;/strong&gt;&lt;/em&gt;']);
    _gaq.push(['_setVar', 'no_report']);
    _gaq.push(['_trackPageview']);
    (function() {
        var ga = document.createElement('script');
        ga.type = 'text/javascript';
        ga.async = true;
        ga.src =
            ('https:' == document.location.protocol ? 'https://ssl' : 'http://www') + '.google-analytics.com/ga.js';
        var s = document.getElementsByTagName('script')[0];
        s.parentNode.insertBefore(ga, s);
    })();&lt;/script
    // ]]&gt;
&lt;meta name=&quot;robots&quot; content=&quot;noindex&quot; /&gt;
&lt;div align=&quot;center&quot;&gt;
    &lt;table width=&quot;100%&quot; border=&quot;0&quot; cellspacing=&quot;0&quot; cellpadding=&quot;0&quot; align=&quot;center&quot;&gt;
        &lt;tbody&gt;
            &lt;tr&gt;
                &lt;td&gt;
                    &lt;div align=&quot;center&quot;&gt;
                        &lt;span style=&quot;font-size: xx-large;&quot;
                            &gt;&lt;span style=&quot;color: #ff6600;&quot;&gt;google analytics cookie excluder&lt;/span&gt;&lt;/span
                        &gt;
                    &lt;/div&gt;
                &lt;/td&gt;
            &lt;/tr&gt;
        &lt;/tbody&gt;
    &lt;/table&gt;
&lt;/div&gt;
</code></pre>
<ol>
<li>In the above code, change _UA-XXXXXXXX-X _to the tracking id.</li>
<li>Place the file anywhere on your site as long as it is under your domain, at the root or in a subdirectory.</li>
</ol>
<h2 id="excluding-yourself">Excluding yourself</h2>
<p>Only carry out the following actions for computers you want to NOT be counted in Google Analytics.</p>
<ol>
<li>In the location field of your browser, open the Html file on your domain.</li>
<li>If everything works, you should see a charcoal background with orange letters saying, &quot;google analytics cookie excluder&quot;.</li>
</ol>
<h2 id="optional-test">Optional Test</h2>
<p>Go to the area of your browser that shows cookies. Search for the value you set in the cookie. If you find it, the value is set.</p>
<h2 id="conclusion">Conclusion</h2>
<p>Now you should be able to remove yourself from the google analytics data.</p>
