<html><head></head><body><h2 id="problem">Problem</h2>
<p>I want to upload multiple files from a specific folder to an AWS S3 bucket.</p>
<h3 id="assumptions">Assumptions</h3>
<p>The S3 bucket name is <strong>test</strong>.</p>
<p>The directory structure is as follows.</p>
<pre><code class="language-txt">documents
|- file_1
|- subdirectory1
|  |- file_1_1
|  |- file_1_2
|  |- subdirectory2
|  |  |- file_1_2_1
</code></pre>
<p>We want to end up with the following S3 objects.</p>
<ul>
<li>s3://test/file_1</li>
<li>s3://test/subdirectory1/file_1_1</li>
<li>s3://test/subdirectory1/file_1_2</li>
<li>s3://test/subdirectory1/subdirectory2/file_1_2_1</li>
</ul>
<h2 id="solution">Solution</h2>
<pre><code class="language-hcl">resource "aws_s3_bucket_object" "test" {
  for_each = fileset("./documents/", "**")
  bucket = "test"
  key = each.value
  source = "./documents/${each.value}"
  etag = filemd5("./documents/${each.value}")
}
</code></pre>
<ul>
<li><strong><em>Line 1</em>:</strong>: Create an S3 bucket object resource.</li>
<li><strong><em>Line 2</em>:</strong>: Use a <strong><em>for_each</em></strong> argument to iterate over the documents returned by the <strong><em>fileset</em></strong> function. <strong><em>for_each</em></strong> identifies each resource instance by its S3 path, making it easy to add/remove files. The fileset function enumerates over a set of filenames for a given path. It uses <strong><em>**</em></strong> as the pattern for a recursive search.</li>
<li><strong><em>Line 3</em>:</strong>: The name of the bucket to put the files in.</li>
<li><strong><em>Line 4</em>:</strong>: The object's name once it's in the bucket. In the example above, it is the same as the path.</li>
<li><strong><em>Line 5</em>:</strong>: the Path to the file to be uploaded.</li>
<li><strong><em>Line 6</em>:</strong>: Triggers an update only if the file changes. The eTag of each object is an MD5 hash of that object.</li>
</ul>
