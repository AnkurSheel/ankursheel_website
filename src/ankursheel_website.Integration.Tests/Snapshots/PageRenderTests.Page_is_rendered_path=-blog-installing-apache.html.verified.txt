<html><head></head><body><h2 id="installing-apache-on-windows">Installing Apache on Windows</h2>
<ul>
<li>Go to the <a href="https://httpd.apache.org/download.cgi">Apache Website</a> and download the installer file.</li>
<li>Make sure Internet Information Services (IIS) is not running.</li>
<li>If IIS is running, you will find the IIS console at Start-&gt;Control Panel-&gt;Administrative Tools-&gt;Internet Services Manager. Shut it down.</li>
<li>Run the installer</li>
<li>In the Server Information dialogue, enter the following
<ul>
<li><strong>Network Domain:</strong> The domain name. For local testing, enter <em>localhost</em>.</li>
<li><strong>Server Name:</strong> The name of the server. For local testing, enter <em>localhost</em>.</li>
<li><strong>Email Address:</strong> The email address where you want to receive email messages about the webserver.</li>
<li><strong>Run Mode:</strong> In most cases, you would want to run Apache as a service.</li>
</ul>
</li>
<li>You can change the installation directory if you click on Custom.</li>
<li>Click Install.</li>
</ul>
<h2 id="installing-and-starting-apache-manually">Installing and Starting Apache Manually</h2>
<p>If you choose to run the apache server manually in the Run Mode, you must install it. This can be done as follows.</p>
<ul>
<li>Run the command prompt as <strong>Administrator</strong>.</li>
<li>Navigate to the bin directory in the Apache install directory and run the following command: <em>httpd.exe -k install</em></li>
<li>Start the server using the Apache Monitor in the taskbar.</li>
</ul>
<h2 id="un-installing-apache">Un-Installing Apache</h2>
<p>Navigate to the bin directory in the Apache install directory and run the following command <em>httpd.exe -k uninstall</em></p>
<h2 id="configuring-apache">Configuring Apache</h2>
<p>To change any settings, edit the httpd.conf file. On Windows, you can access this file through the menu at Start -&gt; Programs -&gt; Apache HTTPD Server -&gt; Configure Apache Server -&gt; Edit the Apache httpd.conf File.</p>
<p>Restart the server for the changes to take effect.</p>
