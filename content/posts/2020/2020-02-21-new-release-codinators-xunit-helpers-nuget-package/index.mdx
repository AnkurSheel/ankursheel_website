I have created [Codinators.XunitHelpers](https://www.nuget.org/packages/Codinators.XunitHelpers/) (a nuget package) to
add some useful unit testing functionality that is missing in xUnit.

The 1st Release adds support for loading test data from a JSON file for xUnit Tests as outlined in the article
[How to load test data from a json file for xUnit tests](./load-test-data-from-a-json-file-for-xunit-tests).

Install it by using the following command

```bash
dotnet add package Codinators.XunitHelpers
```
